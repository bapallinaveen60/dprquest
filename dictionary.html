Variable Dictionary

Search the core scientific parameters of GPM-DPR Level-2 files. Understand their physical meaning, units, and array dimensions.

zFactorMeasured

Units: dBZModule: PRE / Level-1

Raw radar reflectivity factor measured by the spacecraft, uncorrected for attenuation.

This is the initial power return from cloud backscatter received by KuPR and KaPR. Because the microwave pulse loses energy as it penetrates rain, zFactorMeasured is always smaller than or equal to the true reflectivity.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:Z_m = Z_e \cdot A_p
Example Range:12.0 to 55.0 dBZ

zFactorCorrected

Units: dBZModule: SLV

True radar reflectivity factor after correcting for path attenuation and Mie scattering.

The final corrected reflectivity factor representing the true physical backscattering profile of the cloud droplets. It compensates for specific attenuation (absorption + scattering) along the propagation path.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:Z_e = Z_m + PIA
Example Range:12.0 to 65.0 dBZ

precipRate

Units: mm/hModule: SLV

Retrieved physical precipitation rate profile at each range gate.

Calculated at each range gate using local DSD intercept parameters and corrected reflectivity. It gives a 3D volume profile of rainfall intensity.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:R = a \cdot Z_e^b
Example Range:0.0 to 150.0 mm/h

precipRateNearSurface

Units: mm/hModule: SLV

Estimated precipitation rate at the lowest clutter-free gate near the Earth's surface.

Estimates the instantaneous rainfall rate reaching the ground. The solver selects appropriate Z-R coefficients based on whether the CSF module classified the rainfall as stratiform or convective.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:R_{near} = a \cdot Z_{e,cfb}^b
Example Range:0.0 to 120.0 mm/h

precipType

Units: Dimensionless FlagModule: CSF

Classification of precipitation column type (Convective, Stratiform, or Other).

Determined by horizontal texture variation (H-method) and vertical bright band peaks (V-method) to select proper Z-R relationships.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Type \in \{1: Stratiform, 2: Convective, 3: Other\}
Example Range:1 (Stratiform), 2 (Convective)

binClutterFreeBottom

Units: Gate Index (1-176)Module: PRE

Range gate number representing the lowest altitude bin unaffected by ground clutter returns.

Isolates the region contaminated by radar beam mainlobe and sidelobe reflections striking the solid ocean or land surface.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Bin_{cfb} = f(ScanAngle, Elevation)
Example Range:150 (out of 176 gates)

binStormTop

Units: Gate Index (1-176)Module: VER

The highest gate number in the vertical column containing detected precipitation signals above the noise floor.

Identified in the VER module as the topmost boundary of the cloud. A lower gate index signifies a taller storm (convective cloud tops reaching 14-16km).

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Bin_{top} = \min \{i \mid Z_m(i) > NoiseThreshold\}
Example Range:45

heightBB

Units: kmModule: VER

Estimated altitude of the center of the radar bright band (melting layer).

Determined from the localized peak in the vertical reflectivity profile gradient where snow ice crystals melt into liquid rain drops.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:H_{BB} = Altitude(Bin_{BB})
Example Range:3.8 km

widthBB

Units: mModule: VER

The vertical thickness of the melting bright band layer.

Represents the depth of the melting transition layer, determined from the vertical curvature of the radar signal surrounding the bright band peak.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Width = (Bin_{bottom} - Bin_{top}) \cdot 125m
Example Range:350 m

meanDomeDiameter

Units: mmModule: DSD

Mass-weighted mean droplet diameter (Dm) of the raindrop size distribution.

Calculated using the DFR (Dual Frequency Ratio) gap between Ku-band and Ka-band to separate rainfall rate from drop size variance.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:D_m = \int D^4 N(D) dD / \int D^3 N(D) dD
Example Range:0.8 to 3.2 mm

paramNw

Units: m^-3 mm^-1Module: DSD

Normalized intercept parameter (Nw) of the droplet size distribution.

Indicates the count density of droplets per unit volume once normalized by the average size diameter.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:N_w = (4^4 / \pi) \cdot LWC / D_m^4
Example Range:1000 to 24000

pathAttenEstimation

Units: dBModule: SRT

Total column Path Integrated Attenuation (PIA) estimated by the Surface Reference Technique.

Computed by comparing the backscatter reduction of the ocean/land surface in the rain column to adjacent clear-air reference targets.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:PIA_{SRT} = \sigma_{0,ref} - \sigma_{0,meas}
Example Range:0.5 to 38.0 dB

srtReliabilityFlag

Units: Dimensionless FlagModule: SRT

Reliability score of the SRT path integrated attenuation estimate.

Flags whether the underlying ground/sea surface reflection was stable enough to provide a high-confidence reference mirror.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Reliability \in \{1: Reliable, 2: Marginal, 3: Unusable\}
Example Range:1

triggerFlag

Units: Binary FlagModule: TRG

Execution flag indicating whether the column contains valid radar signal above the noise threshold.

Used by the main software thread to bypass empty clear-sky coordinates, maximizing computation efficiency across raw swaths.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Trigger = \exists i \mid Z_m(i) > MinThreshold
Example Range:1 (Process), 0 (Skip)

binLowestTrigger

Units: Gate Index (1-176)Module: TRG

The lowest altitude gate that met the trigger threshold to run the Level-2 pipeline.

Determines the search start altitude for noise check threads, optimizing computational loop lengths.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Bin_{trg} = \max \{i \mid Z_m(i) > MinThreshold\}
Example Range:142

localNoiseLevel

Units: dBmModule: PRE

Estimated instrument receiver thermal noise floor.

Computed from raw signal power at high altitudes (e.g. >18km) where clouds and atmospheric echoes are absent.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Noise = Average(P_{empty})
Example Range:-112.5 dBm

binZeroDegIsotherm

Units: Gate Index (1-176)Module: VER

Estimated gate index representing the freezing level altitude (0°C isotherm).

Identified using temperature profiles from ancillary meteorological models to constrain search boundaries for the bright band detection loop.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Bin_{0C} = f(ModelTemp, Climatology)
Example Range:80

precipWaterCont

Units: g/m³Module: DSD

Liquid water content (LWC) representing the mass density of liquid water per unit volume.

Extracted in the DSD module. It represents the physical mass concentration of atmospheric water droplets inside each grid cell volume.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:LWC = (\pi / 6) \cdot \rho_w \int D^3 N(D) dD
Example Range:0.05 to 4.5 g/m³

attenuationCoeff

Units: dB/kmModule: SLV

Specific attenuation coefficient (k) representing signal decay per kilometer of travel.

Calculated gate-by-gate in the solver. It maps microwave power absorption rates which are integrated to correct measured reflectivity.

Data Specifications
Array Shape:(nScan, nBeam, nBin)
Formula:k = \alpha \cdot Z_e^\beta
Example Range:0.001 to 6.5 dB/km

epsilon

Units: Dimensionless RatioModule: SLV

Adjustment parameter used to scale specific attenuation to match SRT boundary targets.

If the integrated Hitschfeld-Bordan path attenuation does not match the surface reference measurement, epsilon scales the specific attenuation coefficient profile.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:\epsilon = PIA_{solved} / PIA_{SRT}
Example Range:0.85 to 1.15

sigmaZeroMeasured

Units: dBModule: SRT

Measured backscattering cross-section of the Earth's surface inside the rain column.

The surface echo returned to the spacecraft. It is attenuated twice as the pulse passes down and back up through the storm.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:\sigma_{0,meas} = MeasuredPower / CalibrationFactor
Example Range:-18.5 dB

sigmaZeroReference

Units: dBModule: SRT

Reference backscattering cross-section of the Earth's surface under clear-air conditions.

Calculated from clean, non-raining pixels nearby to represent what the surface mirror return should be in the absence of precipitation.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:\sigma_{0,ref} = Mean(\sigma_{0,clear})
Example Range:-10.0 dB

binMeltingLevel

Units: Gate Index (1-176)Module: VER

The top boundary gate index of the melting snow layer.

Extracted in the VER module. It marks where falling snowflakes first begin melting and accumulating water films.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Bin_{melt} = Bin_{BB} - Width/2
Example Range:72

freezingLevel

Units: kmModule: VER

Freezing isotherm height representing the physical boundary between snow and liquid rain.

Used as a threshold boundary in the classification and scattering models to switch from ice-scattering to water-scattering coefficients.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Height_{freezing} = Altitude(Bin_{0C})
Example Range:4.2 km

rainFlag

Units: Binary IndicatorModule: CSF

Flag indicating whether precipitation exists in the vertical column.

Extracted during classification. It filters out columns that only contain trace humidity or high non-precipitating clouds.

Data Specifications
Array Shape:(nScan, nBeam)
Formula:Flag = (Z_{max} > Threshold) ? 1 : 0
Example Range:1 (Rain), 0 (Clear)
